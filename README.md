## 1. 사이트 개요

- 이 사이트는 사관학교 수강생이 1주차 과제로 비전보드와 인생그래프를 작성할 수 있도록 만든 정적 웹사이트입니다.
- 복잡한 개발 환경 없이 HTML 파일만으로 동작합니다.
- GitHub에 변경사항을 올리면 Vercel을 통해 배포됩니다.
- 원격 저장소는 `https://github.com/levigrowth/mfw-school-pages.git`입니다.

-----

## 2. 사이트 구성

Vercel에 배포되는 핵심 파일은 아래 3개입니다.

| 파일 | 설명 |
| --- | --- |
| `index.html` | 홈 화면에서 비전보드와 인생그래프 페이지로 이동합니다. |
| `vision_board.html` | 5년 후 커리어 모습을 작성하고 PDF 또는 HTML 파일로 저장합니다. |
| `life_graph.html` | 나이별 사건과 만족도를 그래프로 정리하고 PDF 또는 이미지 파일로 저장합니다. |

-----

## 3. 저장 방식

이 사이트에서 작성한 내용은 서버에 저장되지 않습니다.

작성 내용은 사용자의 브라우저에만 저장됩니다. 같은 사람이라도 다른 컴퓨터, 다른 브라우저, 시크릿 모드에서는 이전 작성 내용이 보이지 않을 수 있습니다.

중요한 결과물은 반드시 페이지 안의 `인쇄(PDF)`, `이미지 저장`, `파일 저장` 기능으로 따로 보관해야 합니다.

## 4. 수정 방법

### 4.1 Codex로 수정하기

Codex는 코드를 잘 몰라도 자연어로 파일 수정을 요청할 수 있는 도구입니다.

1. 수정하고 싶은 문구나 기능을 정합니다.
2. Codex에서 이 폴더를 엽니다.
3. Codex에게 원하는 수정사항을 자연어로 요청합니다.
4. 수정된 파일을 브라우저에서 열어 확인합니다.
5. 문제가 없으면 GitHub에 올립니다.
6. Vercel 배포 주소에서 최종 확인합니다.

예시 요청:

```text
비전보드 페이지의 제목을 "나의 5년 커리어 비전"으로 바꿔줘.
```

```text
인생그래프 페이지의 저장 안내 문구를 더 쉽게 바꿔줘.
```

```text
홈 화면에서 비전보드와 인생그래프 카드 순서를 바꿔줘.
```

Codex가 파일을 수정한 뒤에는 반드시 브라우저에서 직접 확인해야 합니다.

최종 확인 주소:

https://mfw-school-pages.vercel.app

### 4.2 직접 수정하기

- 홈 화면 문구 수정: `index.html`에서 첫 화면 제목, 카드 설명, 하단 외부 링크를 수정합니다.
- 비전보드 질문 수정: `vision_board.html`에서 입력칸 제목, 안내 문구, placeholder 문구, 버튼 문구를 수정합니다.
- 인생그래프 문구 수정: `life_graph.html`에서 페이지 제목, 버튼 문구, 그래프 안내 문구, 저장/인쇄 관련 문구를 수정합니다.

## 5. Git 사용 방법

### 5.1 Git 설치 확인

처음 사용하는 컴퓨터라면 Git이 설치되어 있어야 합니다.

설치 확인:

```bash
git --version
```

버전 번호가 나오면 설치되어 있는 것입니다.

### 5.2 Git 기본 명령어

변경된 파일 확인:

```bash
git status
```

변경 파일 추가:

```bash
git add .
```

변경 내용 저장:

```bash
git commit -m "수정 내용 설명"
```

GitHub에 올리기:

```bash
git push
```

GitHub 저장소가 Vercel과 연결되어 있다면, `git push` 후 자동으로 배포됩니다.

## 6. 수정 시 주의사항

- 파일 이름을 함부로 바꾸지 않습니다.
- `index.html`, `vision_board.html`, `life_graph.html` 사이의 링크가 끊기지 않았는지 확인합니다.
- 작성 내용은 서버에 저장되지 않습니다.
- 사용자의 작성 내용을 관리자가 볼 수 있는 구조가 아닙니다.
- 큰 수정 전에는 먼저 Git에 커밋하거나 파일을 복사해 백업합니다.
- 배포 후에는 반드시 실제 주소에서 한 번 더 확인합니다.
